# Real-Time Chat App

NestJS, GraphQL, PostgreSQL, Socket.io

## Features

- JWT Authentication
- Event-driven approach for clean architecture
- Real-time messaging (WebSockets)
- Room-based chats
- Message history
- Active user tracking

## Setup

1. Clone repo
2. `npm install`
3. Create `.env` (see `.env.example`)
4. `npm run start:dev`

## Endpoints

- **GraphQL**: `/graphql`
- **WebSocket**: `ws://localhost:3000`
